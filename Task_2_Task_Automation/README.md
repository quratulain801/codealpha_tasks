# Task Automation with Python (Email Extraction)

This project automates the process of extracting email addresses from a text file
using Python.

## Project Description
The script reads a `.txt` file, extracts all email addresses using regular
expressions, and saves the extracted emails into a new file.  
This helps automate a repetitive real-life task efficiently.

## Features
- Reads data from a text file
- Extracts email addresses using regex
- Saves extracted emails to another file
- Simple and beginner-friendly automation

## Key Concepts Used
- Regular Expressions (re)
- File Handling
- Automation using Python

## Technologies Used
- Python

## How to Run
1. Place the input text file in the same directory
2. Run the script:
   ```bash
   python automation.py
